# 🍺 Hop & Barley Shop

**Hop & Barley** — это современный интернет-магазин премиум-класса для домашних пивоваров. Платформа сочетает в себе эстетику крафтовой культуры и передовые технологии веб-разработки.

---

## ✨ Ключевые возможности

* **🎨 Премиальный UI (Glassmorphism)**: Глубокий темный интерфейс с эффектами матового стекла, плавными анимациями и адаптивностью под любые устройства.
* **⚡ Tailwind CSS**: Скоростной интерфейс на базе утилитарных классов с полноценной поддержкой динамической смены тем (Dark/Light).
* **🔍 Smart Catalog**: Продвинутая фильтрация и поиск товаров с использованием AJAX. Результаты обновляются мгновенно без перезагрузки всей страницы.
* **🌍 Мультиязычность (I18N)**: Полная локализация интерфейса и контента (RU/EN). Управление переводами товаров реализовано через `django-modeltranslation`.
* **📦 Современный Backend**: Стабильная работа на **Django 6.0** и **PostgreSQL**, управление зависимостями через современный пакетный менеджер **Poetry**.

## 🚀 Быстрый старт (через Docker)

Убедитесь, что у вас установлены **Docker** и **Docker Compose**.

### 1. Подготовка и запуск
```bash
# Клонируйте репозиторий
git clone <your-repository-url>
cd myshop

# Сборка и запуск контейнеров в фоне
docker-compose up --build -d
```

## 2. Инициализация базы данных
Выполните миграции и загрузите предустановленный каталог товаров:
```bash
# Применение миграций
docker-compose exec web python manage.py migrate

# Загрузка категорий и товаров из фикстур
docker-compose exec web python manage.py loaddata categories.json products.json
```
### 3. Доступ к приложению
Откройте браузер и перейдите по адресу: http://localhost:8000
Панель администратора: http://localhost:8000/shop_admin/

##🌐 Локализация и переводы
Проект поддерживает два языка: Русский и Английский.

Интерфейс: Тексты в шаблонах обернуты в теги {% trans %}. Файлы локализации находятся в директории /locale.

База данных: Каждое поле (например, name, description) имеет версии для обоих языков (например, name_ru, name_en).

Обновление переводов интерфейса:
```bash
# Сбор новых фраз
docker-compose exec web python manage.py makemessages -l en -l ru

# Компиляция (необходимо для отображения изменений)
docker-compose exec web python manage.py compilemessages
```

## 📁 Структура проекта (основное)
/templates — Django шаблоны (включая папку /static_html с архивными макетами).

/products — Приложение каталога товаров и категорий.

/orders — Логика корзины и оформления заказов.

/static — Статические файлы (JS, изображения логотипов и иконок).

## 📝 Разработка локально (без Docker)
Если вы предпочитаете запускать проект напрямую через Poetry:
```bash
poetry install
poetry run python manage.py runserver
```
